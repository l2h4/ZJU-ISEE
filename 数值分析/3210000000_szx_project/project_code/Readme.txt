一、安装
这是通过matlab编写的APP程序，可以通过Approximation_Function.mlapp文件直接在matlab中打开，也可以安装进matlab的Apps中。
方法一：直接在matlab中打开.mlapp文件，点击“运行”即可使用
方法二：在matlab的Apps中，点击“安装App”,选择Approximation_Function.mlappinstall文件进行安装，安装后在我的App中即可使用该App

二、使用
1. 在与Approximation_Function.mlapp文件同一目录下的xydata.xlsx文件中输入数据
2. 在应用中先读取数据，再进行绘制
3. 支持数据乱序输入
4. 支持同一图中的多图像绘制
5. 支持分段函数的颜色区分
（详细的使用及图说明见pdf报告）

三、注意事项
1. 程序从xydata.xlsx文件中读取数据，所以不要移动该文件位置
2. 请避免打乱xydata.xlsx文件中的格式
